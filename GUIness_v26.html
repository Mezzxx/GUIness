        ⊙ HOW TO CONNECT A LOCAL LLM
        

          
OLLAMA (DEFAULT)

          
1. Install Ollama from ollama.com

          
2. Run a model: ollama run llama3

          
3. Enable CORS: OLLAMA_ORIGINS=* ollama serve

          
4. Hit Detect above to auto-populate models

          

          
OPENCLAW

          
1. Run: ollama launch openclaw

          
2. Follow the setup wizard — it configures Ollama at 127.0.0.1:11434

          
3. Leave the URL as default and hit Detect to find your model

          

          

          
OPENCLAW

          
1. Enable the chat completions endpoint in ~/.openclaw/openclaw.json:

          
{"gateway":{"http":{"endpoints":{"chatCompletions":{"enabled":true}}}}}

          
2. Restart: openclaw gateway restart

          
3. Set URL to http://localhost:18789

          
4. Add your gateway token to the Vault as OpenClaw Token

          
5. Hit Detect — agents appear as models (e.g. openclaw:main)

          

          
OTHER OPENAI-COMPATIBLE SERVERS

          
LM Studio, llama.cpp, Jan — set their server URL above and hit Detect
