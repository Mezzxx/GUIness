        ⊙ HOW TO CONNECT A LOCAL LLM
        

          
OLLAMA (DEFAULT)

          
1. Install Ollama from ollama.com

          
2. Run a model: ollama run llama3

          
3. Enable CORS: OLLAMA_ORIGINS=* ollama serve

          
4. Hit Detect above to auto-populate models

          

          
FRIDAY AI AGENT

          
1. Start the Friday backend: cd friday && npm run dev

          
2. Friday runs Express+WS on http://localhost:3777 with Ollama as its default provider

          
3. Set URL to http://localhost:3777

          
4. If Friday requires auth, add your token to the Vault as Friday Token

          
5. Hit Detect — Friday exposes its configured models via OpenAI-compat /v1/models

          

          
OTHER OPENAI-COMPATIBLE SERVERS

          
LM Studio, llama.cpp, Jan — set their server URL above and hit Detect
