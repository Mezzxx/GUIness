# Code of Conduct

## Our Commitment

This project is maintained as open source for anyone to use, learn from, and contribute to. Everyone participating — whether filing issues, submitting pull requests, or joining discussions — is expected to make the experience respectful and constructive for others.

## Expected Behavior

- Be respectful and considerate in communication
- Give and receive feedback graciously
- Assume good intent; ask for clarification before assuming the worst
- Focus on what's best for the project and its users
- Respect differing viewpoints and experience levels

## Unacceptable Behavior

- Harassment, insults, or personal attacks
- Discriminatory language or jokes targeting identity, background, or experience level
- Publishing others' private information without consent
- Trolling, derailing, or sustained disruption of discussions
- Any conduct that would be considered inappropriate in a professional setting

## Scope

This Code of Conduct applies to all project spaces: issues, pull requests, discussions, and any other communication channels associated with GUIness. It also applies when representing the project in public spaces.

## Enforcement

Instances of unacceptable behavior can be reported by contacting the [project maintainer](https://github.com/Mezzxx). All reports will be reviewed and addressed. The maintainer reserves the right to remove, edit, or reject contributions and to temporarily or permanently ban any contributor whose behavior violates this Code of Conduct.
